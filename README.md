# ERPNext ES AEAT

Spanish Tax Authority (AEAT) BOE export module for ERPNext.

## Supported Models

- **111** - IRPF Retenciones (professionals)
- **115** - IRPF Retenciones (arrendadores)
- **130** - IRPF Estimación Directa
- **303** - IVA Trimestral
- **347** - Operaciones >3,050.60€
- **349** - Operaciones Intracomunitarias
- **390** - IVA Anual (resumen)

## Features

- GL Entry-based tax calculation engine
- Official AEAT BOE `.txt` file generation
- Tax Map system (casilla↔cuenta mapping)
- Multi-entity support
